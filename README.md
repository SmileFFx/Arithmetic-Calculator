Funcionalidades

Validação de números e operadores antes da execução do cálculo

Tratamento de erros com blocos try/except

Prevenção de divisão por zero

Permite realizar múltiplos cálculos em sequência

Controle de saída do programa pelo usuário

Como usar

Execute o script no terminal:

python calculadora_aprimorada.py


Digite dois números e escolha o operador desejado (+, -, *, /).

Após exibir o resultado, o programa perguntará se deseja continuar ou encerrar.

Exemplo de execução
Digite um número: 10
Digite outro número: 2
Digite o operador (+, -, *, /): /
O resultado de 10.0 / 2.0 é: 5.0
Deseja sair? (s/n): n

Tecnologias utilizadas

Python 3.x
