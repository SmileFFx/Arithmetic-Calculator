# Calculadora Aprimorada

Calculadora desenvolvida em Python para realizar operações aritméticas básicas (`+`, `-`, `*`, `/`) com tratamento de erros e validação de entradas. O programa foi projetado para ser interativo, robusto e fácil de usar em linha de comando.

---

## Funcionalidades

* Validação de números e operadores antes da execução do cálculo
* Tratamento de erros com blocos `try/except`
* Prevenção de divisão por zero
* Permite realizar múltiplos cálculos em sequência
* Controle de saída do programa pelo usuário

---

## Como usar

1. Execute o script no terminal:

 ```bash
python calculadora_aprimorada.py
 ```
2. Digite dois números e escolha o operador desejado (`+`, `-`, `*`, `/`).
3. Após exibir o resultado, o programa perguntará se deseja continuar ou encerrar.

---

## Exemplo de execução

```
Digite um número: 10
Digite outro número: 2
Digite o operador (+, -, *, /): /
O resultado de 10.0 / 2.0 é: 5.0
Deseja sair? (s/n): n
```
